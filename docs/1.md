```
你上一次真正感到恐惧是什么时候？
不仅是现在，在之后的将来，我也会一次次地询问：
你的恐惧，是真实的吗？
或者换句话说，它基于现实吗？
它很可能是——
吞没房子的烈火，
痛击肠胃的拳头，
刺向咽喉的刀子。
但若它是一个捏造的东西呢——
一只鸟模仿一个无辜者的尖叫，
一个你房间里影子形成的轮廓，
一个你在夜晚幻想出来的怪物。
我想，你脑海里应该已经有它们的画面了吧。
它们远不止这些，我们需要你为我们揭示更多关于它们的奥秘。

你的守灯人，
以斯拉·阿什福德
```
